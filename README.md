# Personal Portfolio Website

This project is a simple and clean personal portfolio website built using **HTML** and **CSS** only.  
It was developed as a practice project to apply web fundamentals such as page structure, layout, styling, and basic responsive design.

##  Project Overview

The portfolio presents a personal profile, skills, and sample projects in a clear and organized way.  
The main goal of this project is to demonstrate understanding of:
- Semantic HTML structure
- CSS layout and styling
- Reusable components
- Clean and readable code

No external libraries or frameworks were used.

##  Technologies Used

- HTML5  
- CSS3  

##  Project Structure

```text
portfolio/
├── index.html        # Home page
├── about.html        # About me page
├── projects.html     # Projects showcase
├── contact.html      # Contact form
├── style.css         # Main stylesheet
└── README.md         # Project documentation



##  Features

- Dark theme design
- Responsive layout for different screen sizes
- Reusable CSS classes (buttons, cards, layout)
- Simple navigation between pages
- Clean and organized file structure

##  Responsiveness

The layout adapts to smaller screens using basic CSS media queries to ensure readability and usability on mobile devices.

##  Purpose of the Project

This project was created for learning and academic purposes, focusing on:
- Practicing HTML and CSS after initial study
- Writing clean, readable, and well-structured code
- Following basic web development best practices
